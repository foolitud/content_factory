# Designing Referrals

## When to use this skill
- Creating a new referral or affiliate program.
- Optimizing existing viral loops or word-of-mouth strategies.
- Calculating incentive sizing and program ROI.

For detailed strategy, examples (Dropbox, Uber, Morning Brew), and recruitment templates, see [ADVICE.md](ADVICE.md).

## Workflow

### 1. Discovery & Goal Setting
Identify the primary objective:
- Customer Referral (social proof)
- Affiliate Program (volume)
- Partner Program (ecosystem)

### 2. Incentive Sizing
Calculate the maximum reward using:
`Max Reward = (Customer LTV × Gross Margin) - Target CAC`

### 3. Loop Implementation
- **Trigger**: Identify the "Aha" moment.
- **Mechanism**: Choose between in-product shares, links, or codes.
- **Conversion**: Design the landing page for referred users.

### 4. Fraud & Guardrails
- Set up email verification.
- Implement delayed payouts for high-value rewards.
- Cap total lifetime rewards per referrer if necessary.

## Instructions
- **Plan-Validate-Execute**: Run a script to check existing user data (LTV/CAC) before proposing incentive sizes.
- **Black Box**: Use analytics scripts to calculate K-factors; run `--help` for parameter details.
